4. В командной строке потестировать следующие команды. Иметь понимание, для чего они используются:

-- man #выдает справку по команде
-- cd #меняет рабочую дирректорию
-- ls #возвращает список файлов в дирректории
-- hostname #возвращает название хоста
-- whoami #возвращает имя пользователя
-- pwd #возвращает путь к рабочей дирректории
-- ip #управление сетевыми устройствами
-- sudo #выполнение компанды от другого пользователя
-- reboot #ребутнуть
-- poweroff #выключить
-- apt или apt-get #управление пакетами

Подключение по ssh по паролю
В терминале:
$ ssh ekaterina@192.168.1.134

Подключение по ssh по ключу
Нужен для более безопасного соединения без ввода пароля.

ssh-keygen  #генерирует ssh ключ
ssh-copy-id ekaterina@192.168.1.134
ssh ekaterina@192.168.1.134
